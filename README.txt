To build
1. Clone Facebook project using git clone "git://github.com/facebook/facebook-android-sdk.git" in a parallel directory as TalkBollywoodAndroid repo.
2. Import facebook project in to eclipse workspace.
3. Ensure facebook project builds.
3. Import the cloned talkbollywood project in to eclipse workspace.
4. Go to talkbollywood project properties->Android->Library add facebook (if not already selected)
5. Build Talkbollywood project
